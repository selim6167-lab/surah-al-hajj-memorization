# Surah Al-Hajj Memorization

A standalone personal memorization app for Surah Al-Hajj.

## Run locally

Open `surah-al-hajj-memorization.html` in a browser, or serve this folder:

```bash
python3 -m http.server 8765
```

Then open:

```text
http://localhost:8765/
```

Progress is stored locally in the browser.
